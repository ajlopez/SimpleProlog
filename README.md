# SimpleProlog

SimpleProlog interpreter in JavaScript.

## Installation

Via npm on Node:

```
npm install simpleprolog
```


## Usage

Reference in your program:

```js
var sf = require('simpleprolog');
```

## Development

```
git clone git://github.com/ajlopez/SimpleProlog.git
cd SimpleProlog
npm install
npm test
```
